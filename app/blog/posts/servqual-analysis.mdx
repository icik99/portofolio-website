**Servqual Analysis** is a dedicated web platform designed to evaluate customer satisfaction for **Indihome PT. Telkom Akses Gorontalo**. By collecting and analyzing feedback through online forms, this website enables the company to generate detailed satisfaction reports for data-driven decision-making.  

![Website Screenshot](/servqual-analysis/1.png)  


## My Role  
![Website Screenshot](/servqual-analysis/2.png)  

As a **System Analyst** and **Frontend Developer**, I contributed to this project by:  

- **System Analysis and Planning:**  
  Designing workflows and user experiences with **Whimsical** as a wireframing tool.  

- **Frontend Development:**  
  Building the website using **Next.js**, **JavaScript**, and **Ant Design (ANTD)** for responsive and professional styling.  

- **Collaboration and API Integration:**  
  Working closely with backend developers using **GitHub** for repository management and **Postman** to ensure seamless API communication.  

- **Deployment:**  
  Successfully deploying the website using **Vercel**, ensuring reliability and accessibility for end users.  

## Key Features  

1. **Customer Satisfaction Survey:**  
   Allows users to fill out satisfaction forms tailored to the services provided by Indihome.  

2. **Report Generation:**  
   Compiles and presents aggregated customer feedback for comprehensive analysis by PT. Telkom Akses.  

3. **Responsive Design:**  
   Ensures seamless usability across various devices and screen sizes.  

## Technologies Used  

- **Framework:** Next.js  
- **Programming Language:** JavaScript  
- **Styling:** Ant Design (ANTD)  
- **Wireframing Tool:** Whimsical  
- **API Collaboration:** Postman  
- **Version Control:** GitHub  
- **Deployment:** Vercel
