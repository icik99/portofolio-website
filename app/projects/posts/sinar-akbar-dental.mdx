**Medical Information System for Sinar Akbar Dental Clinic** is a web-based application designed to support the management of dental clinic operations. The system includes features such as an odontogram, which allows dentists to map and monitor the condition of patients' teeth, improving efficiency and accuracy in providing dental care.

![Website Screenshot](/dental-clinic/2.png)

## My Role

As a **System Analyst** and **Frontend Web Developer**, my responsibilities included:

- **System Analysis and Requirement Gathering:**  
  Collaborating with the client to understand their needs and design a tailored system for the dental clinic.

- **Frontend Web Development:**  
  Developing a responsive and user-friendly interface using **ReactJS**, **JavaScript**, and **TailwindCSS**. This includes implementing the odontogram feature to assist dentists in mapping teeth easily.

- **Client Communication:**  
  Actively engaging with the client to ensure that the developed system aligns with their expectations and operational needs.

- **Agile Development:**  
  Applying **Agile methodology** to deliver the project iteratively, ensuring continuous improvement based on feedback.

- **Collaboration and Code Management:**  
  Utilizing **GitHub** for version control and **Postman** for API testing, ensuring smooth development workflows.

- **Website Testing:**  
  Conducting Blackbox Testing and Usability Testing to identify and address potential issues, guaranteeing a seamless user experience.



## Key Features

1. **Odontogram Mapping:**  
   A feature that allows dentists to visually map and document the condition of each tooth, streamlining treatment planning and patient communication.

2. **Patient Management System:**  
   Enables efficient handling of patient data, including medical records, appointment schedules, and treatment histories.

3. **Responsive Design:**  
   The system is optimized for various devices, ensuring accessibility on desktops, tablets, and mobile phones.



## Technologies Used

- **Frontend Framework:** ReactJS
- **Programming Language:** JavaScript
- **Styling:** TailwindCSS
- **API Testing:** Postman
- **Version Control:** GitHub
