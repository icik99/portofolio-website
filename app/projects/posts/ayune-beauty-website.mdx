**Ayune Beauty** is a beauty website that provides a curated list of beauty products from well-known brands, offering product recommendations and enabling users to directly consult with certified beauty doctors. The platform serves as a comprehensive hub for beauty-related advice, product information, and personalized recommendations.


## My Role  

As a **Backend Developer** for this project, I was responsible for:

- **API Development:**  
![Website Screenshot](/ayune/1.png)
  Utilizing **ExpressJS** with **JavaScript** to create and manage backend services, ensuring smooth communication between the front-end and the database.

- **Database Management:**  
  Implementing **Sequelize ORM** with **MySQL** for efficient database management and smooth interaction between the server and data.

- **Real-Time Consultation Feature:**  
  Integrating **Socket.IO** for real-time chat consultation, enabling users to communicate directly with certified beauty doctors.

- **API Testing:**  
  ![Website Screenshot](/ayune/2.png)
  Using **Postman** for API testing to ensure robust and reliable endpoints.

- **Collaboration and Code Management:**  
  Managing the project repository and code collaboration using **GitHub**.

## Key Features  

1. **Beauty Product Catalog:**  
   Provides a wide range of beauty products from top brands, allowing users to explore different options and find what suits their needs.

2. **Personalized Recommendations:**  
   Offers product recommendations based on user preferences and previous interactions, helping users discover the best products for their beauty routines.

3. **Real-Time Chat Consultation:**  
   Users can consult directly with certified beauty doctors via **Socket.IO**, getting expert advice in real time.

4. **Secure and Scalable Backend:**  
   Developed a robust backend with **ExpressJS**, **Sequelize ORM**, and **MySQL** to ensure scalability and maintain high performance even with increasing traffic.

## Technologies Used  

- **Backend Framework:** ExpressJS  
- **Programming Language:** JavaScript  
- **Database:** MySQL (with Sequelize ORM)  
- **Real-Time Communication:** Socket.IO  
- **API Testing:** Postman  
- **Version Control:** GitHub  

## Impact  

This project demonstrates my ability to build scalable and user-friendly backend solutions, particularly in the context of real-time communication and database management. By integrating live consultation features and recommending personalized beauty products, Ayune Beauty provides a seamless and interactive experience for users seeking expert advice and the best products for their beauty needs.
