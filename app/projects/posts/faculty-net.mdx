**Faculty Net** is a web-based platform specifically designed to monitor and manage Wi-Fi networks for the Faculty of Economics and Business, Business Communication, and the Manterawu Building. This solution provides detailed insights into network performance, helping stakeholders optimize network usage effectively.  

![Website Screenshot](/faculty-net/1.png)  

## My Role  

As a **System Analyst** and **Programmer** for this project, I was responsible for:  

- **Integration of PRTG API:**  
  Enabling real-time monitoring and analysis of network conditions through seamless API integration.  

- **Web Development:**  
  Utilizing **Next.js**, **JavaScript**, **ExpressJS**, and **TailwindCSS** to create a responsive and user-friendly interface for efficient network monitoring.  

- **Collaboration and Code Management:**  
  Leveraging **GitHub** for version control and repository management to ensure seamless teamwork and project integrity.  

## Key Features  

1. **Real-Time Network Monitoring:**  
   Provides up-to-date insights into Wi-Fi performance, including bandwidth usage, connection stability, and device activity.  

2. **Analytics Dashboard:**  
   Offers a user-friendly interface for analyzing historical data to identify trends and potential areas for improvement.  

3. **Optimized for Faculty Use:**  
   Tailored specifically for the unique requirements of the targeted faculties and buildings.  

## Technologies Used  

- **Framework:** Next.js  
- **Programming Language:** JavaScript  
- **Styling:** TailwindCSS  
- **Backend:** ExpressJS  
- **Monitoring API:** PRTG API
